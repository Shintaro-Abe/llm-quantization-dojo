# 02. 無料ハンズオン実行環境の調査

調査日: 2026-06-13 ／ 手段: Gemini（Web検索裏取り）

## 結論（3行）

- **Google Colab（無料T4）** と **Kaggle Notebooks（無料T4×2＝32GB）** が学習の主力。7B〜13Bの量子化・QLoRAが現実的に回せる。
- **Hugging Face Spaces (ZeroGPU)** は短時間の推論デモ向き。学習用の長時間処理には不向き。
- **ローカル実行**は llama.cpp（GGUF）でCPUでも7B〜13B推論が可能。GGUF変換はVRAMよりRAMが効く。

## 1. 無料GPU環境の比較

| 環境 | 主なGPU(VRAM) | 利用枠/制限 | 1セッション | 特徴・注意 |
| :-- | :-- | :-- | :-- | :-- |
| **Google Colab(無料)** | NVIDIA T4(約15GB) | 動的（保証なし） | 最大12時間目安 | 混雑時に使えないことあり。無操作で切断。リソース非保証 |
| **Kaggle Notebooks(無料)** | T4×2(32GB) / P100(16GB) | 30時間/週 | 最大12時間 | 2×T4で高VRAM。Colabより安定傾向 |
| **Hugging Face Spaces (ZeroGPU)** | 大容量GPU(H200級) | 1日あたりの利用上限あり（短時間） | 1リクエスト数十秒級 | 推論デモ向け。学習には不向き。具体の秒数/枠は**不確実**（要公式確認） |
| **Lightning AI(無料)** | T4 / L4 など | 月次の無料クレジット制 | 手動再起動 | GPUを選べる。クレジット消費に注意。具体量は**不確実** |
| **Paperspace Gradient(無料)** | 旧世代GPU(例:8GB級) | 可用性依存 | 6時間目安 | 無料GPUは混雑しがち。永続ストレージ小 |

> 値は調査時点の目安。**各社の規約・GPU割当は変動が大きい**ため、利用時に公式ページで再確認すること。

## 2. 量子化に必要なVRAM/RAMの目安（無料枠で動くか）

### 2.1 QLoRA（ファインチューニング）
| モデル | 目安VRAM | 無料枠での現実性 |
| :-- | :-- | :-- |
| 7B | 約7〜10GB | Colab/Kaggle/Lightningで可 |
| 13B | 約12〜16GB | Colab/Kaggle/Lightningで可（KaggleのT4×2が快適） |
| 30B | 約20〜28GB | 無料枠では厳しい |
| 70B | 約40〜48GB | 無料枠では不可 |

### 2.2 GPTQ/AWQ（4bit量子化済みモデルの推論）
| モデル | 目安VRAM | 無料枠での現実性 |
| :-- | :-- | :-- |
| 7B | 約5〜8GB | 主要な無料枠で可 |
| 13B | 約9〜12GB | Colab/Kaggle/Lightningで可 |
| 30B | 約20〜25GB | ギリギリ/クレジット消費で可 |
| 70B | 約35〜48GB | 無料枠では不可 |

### 2.3 GGUF変換
- 変換は主に **CPU + RAM** で行い、VRAMはほぼ不要。
- 目安RAM: 7B≈16GB / 13B≈32GB / 30B≈64GB / 70B≈128GB超。
- 無料枠ではRAM不足になりやすく、7B〜13Bの変換が現実的。

> VRAM/RAM 目安は学習・コミュニティ情報に基づく**概算**。モデル構造・量子化設定で変わる（不確実）。

## 3. ローカル実行（CPU/小型GPU・llama.cpp）の現実性

- **CPU推論(llama.cpp, Q4_K_M目安)**: 7B≈5GB RAM / 13B≈9.5GB / 30B≈22GB / 70B≈43GB。速度はメモリ帯域依存（7Bで概ね10〜15 tok/s、70Bで1〜2 tok/sと低速）。
- **小型GPU**: 8GB VRAM級（例: RTX 3050/4060 8GB）で7B〜8Bの4bit推論が快適。4GBでは3B以下が現実的。
- `--flash-attn` やKVキャッシュ量子化でVRAMを節約可能。

## 学習ツールへの示唆

- 初学者の最大の障壁は「環境構築（CUDA/依存関係/GPU確保）」。**Colab/Kaggleの“ボタン1つで開く”導線**を用意すると挫折を防げる。
- 教材は **7B（やそれ以下のTinyモデル）×4bit** を基準に設計すれば、無料枠で完走できる。
