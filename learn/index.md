# LLM量子化道場

初学者AIエンジニアが **LLM量子化を「迷わず学び、学習後に実務で使える」** ことを目指す、完全無料の個人学習ツールです。

## このサイトでできること

- **座学**: 各量子化手法の「何のため・いつ使う」を平易に理解する
- **ハンズオン**: Google Colab（無料GPU）で、環境構築なしにコードを最後まで実行する
- **公式情報への導線**: 一次情報（公式ドキュメント・論文）へ正しくたどり着く
- **進捗管理**: GitHub Issues / Projects で学習を記録する

## 学習の進め方（章＝4点セット）

各章は次の4ステップで構成します。上から順に進めれば迷いません。

1. **概念**（このサイトの章ページ）— 何のため・どう使うかを薄く理解する
2. **公式リンク集**（references）— 一次情報で裏取りする
3. **ハンズオン Notebook**（Colab）— 実際に動かして体感する
4. **理解度確認**（GitHub Issue）— 学んだことをタスク化して記録する
